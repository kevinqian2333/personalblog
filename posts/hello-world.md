## 关于这个博客

大家好！欢迎来到我的个人博客。这个博客是我基于 **Next.js 16** 和 **React 19** 构建的，采用了当下非常流行的**磨砂玻璃（Glassmorphism）设计风格**。

## 技术栈

这个博客使用了以下技术：

- **Next.js 16** - React 全栈框架，支持 App Router
- **React 19** - 最新的 React 版本
- **TypeScript** - 类型安全的 JavaScript
- **Tailwind CSS v4** - 实用优先的 CSS 框架
- **Framer Motion** - 强大的动画库

## 设计理念

### 磨砂玻璃风格

整个博客采用磨砂玻璃（Glassmorphism）设计，核心特点包括：

- 半透明背景配合 backdrop-filter 模糊效果
- 细边框增强层次感
- 柔和阴影营造悬浮感
- 渐变背景色流动

### 深空主题

暗黑模式下，背景呈现出深邃的太空感，配合"流萤飞舞"的粒子动画效果，营造出梦幻的氛围。亮色模式下则切换为樱花飘落的唯美场景。

## 功能特点

1. **明暗主题切换** - 一键切换，动画丝滑流畅
2. **全局音乐播放器** - 支持网易云音乐API
3. **Markdown 文章渲染** - 支持代码高亮和数学公式
4. **响应式设计** - 完美适配桌面端和移动端
5. **文章搜索** - 快速定位你感兴趣的内容
